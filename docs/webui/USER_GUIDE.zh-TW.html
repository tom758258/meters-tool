Meters Tool WebUI 使用指南

本指南適用於已取得建置完成的 WebUI 啟動器，並使用它擷取與記錄支援之數位萬用電表測量資料的操作員。本指南不包含開發人員細節，著重於一般的測量工作流程。

WebUI 的功能

WebUI 會在本機開啟瀏覽器頁面，用於設定與監控採集執行。其功能包括：

  • 尋找連線的 VISA 儀器。
  • 一次啟動一個測量執行。
  • 將讀數記錄至 CSV 檔案。
  • 在執行處於作用中狀態時，顯示最新讀數與近期讀數。
  • 在所選觸發模式需要時，傳送軟體觸發。
  • 停止目前執行並開啟已完成的 CSV 檔案。

WebUI 執行在可存取該儀器的同一台 Windows 電腦上。它並非雲端服務。

啟動 WebUI

在一般情況下，雙擊發行套件或本機建置中提供的 WebUI 啟動器：

meters-tool-webui-launcher.exe

發行套件資料夾可能包含帶有版本號的啟動器名稱，例如：

meters-tool-webui-launcher-<version>.exe

在啟動器視窗中：

  1. 保持選擇 Use default port 8767（使用預設連接埠 8767），除非該連接埠已被佔用。
  2. 按一下 Start。
  3. 等待瀏覽器開啟。啟動器會在您的電腦上啟動本機 WebUI 伺服器，並自動為您開啟瀏覽器網頁。
  4. 在您使用完 WebUI 後，按一下啟動器中的 Quit（結束）以關閉。

若瀏覽器未自動開啟，請手動開啟此網址：

http://127.0.0.1:8767/

開發人員或自原始碼簽出（source checkout）的使用者，應參閱 WebUI README 以取得終端機命令、驗證以及建置細節。

畫面總覽

WebUI 是一個本機採集主控台。主要區域如下：

  • Device / Resource（裝置 / 資源）：儀器位址、上次掃描的實體資源、Scan Device（掃描裝置）按鈕以及 Device options（裝置選項）齒輪。此區域預設展開，可折疊為簡短的資源/模型摘要。
  • Run Setup（執行設定）：CSV 輸出路徑與執行次數設定。
  • Measurement（測量）：測量類型與相關選項。
  • Trigger（觸發）：觸發模式與觸發相關選項。
  • Status（狀態）：目前執行狀態、已擷取的樣本數、錯誤數、CSV 路徑與記錄（log）。
  • Live data（即時資料）：最新讀數、趨勢圖表、近期樣本以及所選樣本的詳細資訊。

瀏覽器語言

使用右上角永久顯示的地球與文字按鈕，可在英文（English）與繁體中文之間進行切換。按鈕會顯示目標語言名稱：英文版顯示 繁體中文，而繁體中文版顯示 English。

首次載入網頁時，會優先使用已儲存的有效語系（locale），其次為瀏覽器偏好語言，最後為英文。手動選擇的語系會儲存於 meters-tool.webui.locale 中；自動偵測的瀏覽器語言不會自動儲存。切換是即時生效的，不需要重新整理網頁，也不會傳送請求至背景執行期或 API。目前表單數值、執行中狀態、面板、狀態記錄、選取的即時資料、圖表設定、資源掃描與支援摘要皆會保持原樣。未知的診斷文字會保持未翻譯狀態。

在繁體中文下，測量選項控制項會顯示 自動量程（Auto range）；摘要則使用較短的 自動量程。在一般的桌上型電腦寬度下，「選填」標記會保留在欄位標題旁（例如 AC 濾波器與電流端子），並在非常窄的螢幕上自然換行。

首次執行

請使用以下工作流程進行基本的立即測量：

  1. 開啟 Keysight 34460A 或 34461A 的電源，並將其連接至電腦。
  2. 啟動 WebUI。
  3. 按一下 Scan Device。
  4. 在 VISA resource 中選擇或複製偵測到的 VISA 資源。
  5. 在 Device options 中將 Expected model 保持為 Auto-detect（自動偵測），除非您需要強制指定 34460A 或 34461A。
  6. 選擇測量類型，例如 DC 電壓或 DC 電流。
  7. 在 Run Setup 中選擇 CSV 位置。使用 Select（選擇）以選擇資料夾並產生帶有時間戳記的 CSV 路徑。
  8. 將觸發模式保持在 immediate／預設（立即）模式，除非您特別需要軟體或外部觸發。
  9. 檢查醒目顯示的設定。
  10. 按一下 Start（開始）。
  11. 觀察 Captured（已擷取）、Status（狀態）與 Live data（即時資料）。
  12. 當需要結束執行時，按一下 Stop（停止）。
  13. 執行停止後，按一下 Open CSV（開啟 CSV）以開啟已完成的 CSV 檔案。

一次只能有一個執行處於作用中狀態。啟動新執行會清除前一次執行所顯示的近期樣本。

選擇觸發模式

對於最簡單的工作流程，請使用立即模式。儀器會在執行一開始時便取得讀數。

當執行需要等待操作員在 WebUI 中手動觸發時，請使用軟體觸發模式。在執行開始後，按一下 Trigger（觸發）以傳送每次的軟體觸發。

僅在已連接實體觸發訊號並為儀器設定好時，才使用外部或硬體觸發模式。硬體觸發逾時是一種保護性的重新配置狀態，不一定代表測量失敗。

除非測量程序要求，否則請勿變更觸發時間、觸發延遲、NPLC、自動量程、自動歸零、VM Comp 或電流端子設定。

設定參考

有關引數限制與有效值，請參閱 CLI 驗證的引數限制 部分。WebUI 會在將表單傳送至背景進行驗證之前，限制可接受的輸入值。

即時資料圖表縮放

趨勢圖表可自訂 Y 軸的縮放。共有三種模式：

  • Auto absolute（自動絕對）：根據可見的近期樣本的實際最小值與最大值自動縮放。這最適合觀測真實的測量範圍。例如，若樣本在 4.9998 V 至 5.0004 V 之間，圖表範圍將基於這些實際值。離群值可能會重新縮放圖表，使微小的變化看起來更平坦。
  • Manual span（手動跨度）：以第一個數值樣本為中心，並加上由操作員輸入的固定正數跨度（span）。該跨度使用原始測量單位：V 代表伏特，A 代表安培，Ohm 代表歐姆，Hz 代表赫茲，s 代表秒。例如對於電壓，0.01 代表 0.01 V，而非 0.01 mV。若第一個樣本是 5.000 V，手動跨度為 0.010 V，圖表會顯示 4.990 V 到 5.010 V。超出 基準值 ± 跨度 的數值將被限制在圖表邊界內，因此第一版不會顯示獨立的剪裁指示器。
  • Range step（範圍階梯）：使用手動選取的 Range 作為圖表顯示跨度。此模式僅在 Auto Range 停用且選擇了手動量程時可用。它不反映儀器實際自動選擇的量程，因為當自動量程啟用時，WebUI 並不知道硬體的實際量程。若第一個樣本是 5.000 V，選取的手動量程為 0.010 V，圖表會顯示 4.990 V 到 5.010 V。超出 基準值 ± 所選手動量程 的數值會被限制在圖表邊界內。與其他圖表縮放模式一樣，Range step 僅影響瀏覽器圖表顯示，不會變更儀器設定、SCPI 命令、CSV 輸出或記錄的值。

CSV 輸出

在 Run Setup 中顯示的 CSV 路徑是按一下 Start 時將使用的檔案。

Select 會在執行 WebUI 的 Windows 電腦上開啟資料夾選取器。選擇資料夾後，WebUI 會在該資料夾中填入具有時間戳記的 CSV 檔案路徑。您可以在按一下 Start 之前手動編輯該路徑。

Open CSV 在執行完成且有 CSV 路徑時可用。它會使用 Windows 預設應用程式開啟上一次完成執行的 CSV 檔案。在執行處於作用中狀態時，此按鈕會被停用。

停止與結束

在瀏覽器中按一下 Stop 可停止目前的採集執行。執行停止後，WebUI 會保持最新讀數可見，以便您進行檢視。

在啟動器視窗中按一下 Quit 可停止本機 WebUI 伺服器並關閉啟動器。僅關閉瀏覽器分頁不一定會停止伺服器。

若您是從開發人員終端機而非啟動器執行 WebUI，請參閱 WebUI README 以取得關閉細節。

常見問題

瀏覽器未開啟

請手動開啟此網址：

http://127.0.0.1:8767/

若網頁仍未載入，請返回啟動器並檢查其是否顯示啟動錯誤。

啟動器顯示連接埠已被佔用

另一個程式已在使用選定的連接埠。若另一個 WebUI 已經在該連接埠上執行，啟動器會自動開啟它。若該連接埠被其他項目佔用，請選擇不同的連接埠或關閉該程式。

Scan Device 找不到任何裝置

請檢查：

  • 儀器是否已開啟電源。
  • 是否已連接 USB/LAN/GPIB 訊號線。
  • VISA 驅動程式是否能看到該儀器。
  • 是否有其他程式佔用了儀器連線。

您仍可以手動輸入已知的 VISA 資源字串。

Scan Device 無法推斷模型

將 Expected model 保持為 Auto-detect 並按一下 Start；後端會重新執行 IDN 預檢。只有在您希望連線儀器不是 34460A 或 34461A 時讓 Start 明確失敗，才需要指定模型。

Start 顯示選擇的模型與 IDN 不符

從 Device options 中選擇訊息中命名的模型，或將 Expected model 改回 Auto-detect。這表示連線儀器的 IDN 與所選的 WebUI 預期模型不符。

Start 被阻擋

確認已填寫 VISA resource，且醒目顯示的設定均為有效值。Status 記錄會顯示需要注意的設定。

Open CSV 被停用

Open CSV 會一直被停用，直到執行停止且已完成的 CSV 路徑可用為止。在執行處於作用中狀態時，它也會保持停用。

硬體觸發執行似乎在等待

外部觸發模式需要等待實體觸發訊號。若遺漏觸發訊號，執行可能會依據所設定的逾時行為繼續等待或重新進入待命狀態。

操作員安全注意事項

  • 在測量電流之前，確認儀器輸入接線與電流端子。
  • 在檢查新設定時，請先使用立即模式。
  • 除非需要固定量程，否則請保持自動量程啟用。
  • 將外部觸發接線與極性視為測量設定的一部分。
  • 在可行情況下，請在斷開儀器連接前停止執行。
