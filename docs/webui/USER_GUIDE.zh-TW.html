Keysight Logger WebUI 使用者指南

本指南適用於收到已建置之 WebUI 啟動器，並使用它來記錄 Keysight 34461A 測量數據的操作人員。本指南避開了開發人員細節，專注於標準的測量工作流程。

WebUI 的功能

WebUI 會啟動一個本機瀏覽器頁面，用於設定與監控數據擷取作業 (acquisition run)。它的功能包含：

  • 尋找已連線的 VISA 儀器。
  • 每次啟動一次測量作業。
  • 將讀值記錄到 CSV 檔案中。
  • 在作業執行期間顯示最新與最近的讀值。
  • 當所選的觸發模式需要時，發送軟體觸發訊號。
  • 停止目前的作業並開啟已完成的 CSV 檔案。

WebUI 執行於與儀器連接的同一台 Windows 電腦上。這不是一項雲端服務。

啟動 WebUI

一般使用情況下，請按兩下執行發布版本或本機建置所提供的 WebUI 啟動器：

keysight-logger-webui-launcher.exe

發布資料夾可能包含帶有版本號的啟動器名稱，例如：

keysight-logger-webui-launcher-<version>.exe

在啟動器視窗中：

  1. 除非該連接埠已被佔用，否則請保持選取 Use default port 8767 (使用預設連接埠 8767)。
  2. 點擊 Start (開始)。
  3. 等待瀏覽器開啟。啟動器會在電腦上啟動一個本機 WebUI 伺服器並自動為您開啟瀏覽器頁面。
  4. 使用完 WebUI 後，點擊啟動器中的 Quit (退出)。

如果瀏覽器沒有自動開啟，請手動開啟此網址：

http://127.0.0.1:8767/

開發人員或從原始碼簽出 (source-checkout) 的使用者，應參閱 WebUI README 以取得終端機指令、驗證與建置的詳細資訊。

畫面總覽

WebUI 是一個本機的擷取控制台。主要區域包含：

  • VISA resource (VISA 資源)：作業將使用的儀器位址。
  • Live resource (即時資源)：掃描偵測到的儀器。
  • Scan Device (掃描裝置)：搜尋已連線的可用儀器。
  • Run Setup (作業設定)：CSV 輸出路徑與執行次數設定。
  • Measurement (測量)：測量類型與相關選項。
  • Trigger (觸發)：觸發模式與觸發相關選項。
  • Status (狀態)：目前的作業狀態、擷取到的樣本數、錯誤、CSV 路徑與日誌。
  • Live data (即時數據)：最新讀值、趨勢圖、最近樣本與選定的樣本詳細資訊。

首次執行

進行基本的立即測量，請採用以下流程：

  1. 開啟 Keysight 34461A 電源並將其連接至電腦。
  2. 啟動 WebUI。
  3. 點擊 Scan Device (掃描裝置)。
  4. 選取或將偵測到的 VISA 資源複製到 VISA resource 欄位中。
  5. 選擇測量類型，例如直流電壓 (DC voltage) 或直流電流 (DC current)。
  6. 在 Run Setup 中，選擇 CSV 檔案位置。使用 Select (選擇) 挑選一個資料夾，系統會產生帶有時間戳記的 CSV 檔案路徑。
  7. 除非特別需要軟體或外部觸發，否則請將觸發模式保持在立即/預設模式 (immediate/default mode)。
  8. 確認反白顯示的設定。
  9. 點擊 Start (開始)。
  10. 觀察 Captured (已擷取)、Status (狀態) 及 Live data (即時數據)。
  11. 當需要結束作業時，點擊 Stop (停止)。
  12. 作業停止後，點擊 Open CSV (開啟 CSV) 以開啟已完成的 CSV 檔案。

一次只能執行一個作業。啟動新作業將會清除畫面上前次作業的最近樣本。

選擇觸發模式

最簡單的工作流程請使用立即模式 (immediate mode)。作業一啟動，儀器就會立即讀取數據。

當作業需要等待操作人員在 WebUI 執行動作時，請使用軟體觸發模式 (software trigger mode)。作業啟動後，點擊 Trigger (觸發) 來發送每次的軟體觸發訊號。

只有在已連接物理觸發訊號並為儀器完成設定時，才使用外部 (external) 或硬體 (hardware) 觸發模式。硬體觸發超時 (timeout) 是一種保護性的重新準備 (re-arm) 條件，並不自動代表測量失敗。

除非測量設定有此需求，且操作人員了解對儀器造成的影響，否則請勿更改觸發時序 (trigger timing)、觸發延遲 (trigger delay)、NPLC、自動換檔 (Auto Range)、自動歸零 (Auto Zero)、電壓測量補償 (VM Comp) 或電流端子 (current terminal) 的設定。

設定參考

WebUI 會在啟動作業前驗證各項設定。如果 Start 被封鎖無法點擊，請閱讀狀態日誌 (Status log) 並調整其指出的欄位。

VISA resource (VISA 資源) 是作業將使用的儀器位址。建議使用透過 Scan Device 找到的資源，或手動輸入由操作人員提供的已知 VISA 資源。當可能連接多台儀器時，請勿用猜測的方式填寫資源。

Live resource (即時資源) 顯示最後一次掃描的結果。請先用它來確認是哪台儀器回應，然後再複製或選取資源以進行作業。

CSV path (CSV 路徑) 是讀值寫入的位置。請使用 Select 選擇資料夾並讓 WebUI 產生帶有時間戳記的檔案名稱，或在點擊 Start 前手動輸入特定的檔案路徑。

執行次數 (Run count) 與樣本限制 (sample limit) 欄位用來限制作業的持續時間。在驗證接線、測量類型與觸發行為時，請為新的設定保持上限限制。

Measurement type (測量類型) 用來選擇 34461A 要測量的項目：直流或交流電壓、直流或交流電流、直流電壓比 (DC voltage ratio)、頻率、週期，或是 2 線/4 線電阻。在啟動作業前，請確保這與儀器的實體接線相符。

Auto Range (自動換檔) 讓儀器自行選擇測量範圍。除非測量程序要求固定範圍，否則首次執行時請保持啟用。

當 Auto Range 停用時會使用手動範圍 (Manual range) 欄位。請選擇一個能安全涵蓋預期訊號的範圍。

NPLC 控制直流與電阻測量的積分時間。較高的數值速度較慢，但可能更穩定。交流、頻率與週期模式則會使用 AC filter 設定。

Auto Zero (自動歸零) 控制直流與電阻測量的偏移處理。它可以提高精確度，但可能會減慢讀取速度。除非測量程序要求更改，否則請保持在正常的設定值。

AC filter 適用於交流電壓、交流電流、頻率與週期。交流電壓與交流電流選擇 Keep current setting 時，不會改變儀器目前的 filter 設定；頻率與週期預設選取 20 Hz。

Gate time 僅適用於頻率與週期，預設為 0.1 s。

量測選項中的 Timeout 僅適用於頻率，預設為 Auto。週期會隱藏此控制項，且不會送出 timeout 指令。

Current terminal (電流端子) 適用於電流測量。啟動作業前，請確認實體導線已連接至相符的電流端子。

直流電壓比 (DC voltage ratio) 與 VM Comp 設定是特殊的測量控制選項。只有當測試設定明確要求進行比值測量或電壓測量補償時才使用。

Trigger mode (觸發模式) 控制採樣的時間點。立即模式 (Immediate mode) 是最簡單的首選。軟體模式 (Software mode) 會等待 WebUI 的 Trigger 按鈕。外部或硬體觸發模式則需要實體的物理觸發訊號。

Trigger delay (觸發延遲) 設定觸發後要等待多久才進行測量。除非外部設定需要延遲，否則請保持不變。

Trigger timeout (觸發超時) 控制觸發工作流程在進入保護性超時機制前的等待時間。只有在測量設定有意等待更長時間時，才調高此值。

External trigger slope (外部觸發斜率) 用於選擇實體觸發的邊緣 (edge)。請與連接到儀器的訊號源保持一致。

CSV 輸出

在 Run Setup 中顯示的 CSV 路徑就是當點擊 Start 時將寫入數據的檔案。

Select (選擇) 會在執行 WebUI 的 Windows 電腦上開啟一個資料夾選擇器。選擇資料夾後，WebUI 會在該資料夾內填入帶有時間戳記的 CSV 檔案路徑。您也可以在點擊 Start 之前手動編輯該路徑。

只有在已完成的作業且具有 CSV 路徑時才能使用 Open CSV (開啟 CSV)。它會使用 Windows 的預設應用程式開啟上一次完成作業的 CSV。在作業執行期間，此按鈕會處於停用狀態。

停止與退出

在瀏覽器中使用 Stop (停止) 來停止目前的擷取作業。WebUI 會在作業停止後保留畫面上最新的讀值，以供您檢視。

在啟動器視窗中使用 Quit (退出) 來停止本機 WebUI 伺服器並關閉啟動器。僅關閉瀏覽器分頁不一定會停止伺服器。

如果您是從開發人員終端機而非啟動器執行 WebUI，請參閱 WebUI README 以取得關閉細節。

常見問題

瀏覽器未開啟

請手動開啟此網址：

http://127.0.0.1:8767/

如果頁面仍然無法載入，請回到啟動器並檢查是否顯示啟動錯誤。

啟動器顯示連接埠已被佔用

所選的連接埠已由另一個程式使用中。如果有另一個 WebUI 已經在該連接埠上執行，啟動器會直接開啟它。如果該連接埠被其他不同的程式佔用，請選擇另一個連接埠或關閉該程式。

Scan Device (掃描裝置) 找不到任何裝置

請檢查：

  • 儀器電源已開啟。
  • 已經接上 USB/LAN/GPIB 連線。
  • VISA 驅動程式能偵測到該儀器。
  • 沒有其他程式正在佔用該儀器的連線。

您仍然可以手動輸入已知的 VISA 資源。

無法點擊 Start (Start 被封鎖)

請確保已填寫 VISA resource，且反白顯示的設定皆具有有效值。狀態日誌 (Status log) 會顯示需要注意的設定。

Open CSV 按鈕被停用

Open CSV 將被停用，直到作業停止並產生完整的 CSV 路徑為止。在作業執行期間，它也會保持停用狀態。

硬體觸發作業似乎卡在等待狀態

外部觸發模式會等待物理的觸發訊號。如果沒有收到觸發訊號，作業會根據設定的超時 (timeout) 行為繼續等待或重新準備 (re-arm)。

操作人員安全注意事項

  • 在測量電流之前，請先確認儀器的輸入接線與電流端子。
  • 檢查新的設定時，請先使用立即模式 (immediate mode)。
  • 除非需要固定範圍，否則請保持啟用自動換檔 (Auto Range)。
  • 請將外部觸發的接線與極性視為測量設定的一部分。
  • 在可行的情況下，請在斷開儀器連接前先停止測量作業。
